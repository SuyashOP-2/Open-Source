### SETUP
In the project directory, you can run:
- `npm install # install dependencies`
- `npm run serve # run project in development mode`
- `npm run build # deploy project to production`
- `npm test # run tests`
### TOOLS USED
- npm 10.2.4
- node v20.9.0
